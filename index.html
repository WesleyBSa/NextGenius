Sobre o Projeto

Conheça nossa iniciativa de democratização do acesso à tecnologia e transformação social através da educação em programação.

Formando a próxima geração de desenvolvedores

O NextGenius é um projeto educacional sem fins lucrativos que visa capacitar estudantes da rede pública de ensino através do aprendizado de programação e tecnologias digitais.

Acreditamos que o acesso à educação tecnológica de qualidade é um direito de todos, e que o conhecimento em programação pode abrir portas para novas oportunidades profissionais e pessoais.

O que Ensinamos

Nossos cursos são projetados para desenvolver habilidades técnicas relevantes e preparar os alunos para os desafios do mercado de trabalho.

Iniciante

Introdução à Programação com PYTHON

⏱️ 3 semanas
📚 Iniciante

Explore a lógica de programação escrevendo seus primeiros scripts em Python: crie variáveis, controle fluxos com condicionais e loops, aprenda a modularizar código com funções e desenvolva o raciocínio necessário para qualquer linguagem.

Tópicos abordados:

Variáveis Estruturas de decisão Loops Funções Entrada/Saída
Iniciante

Fundamentos de HTML5 e CSS3

⏱️ 4 semanas
📚 Iniciante

Aprenda a montar a estrutura semântica de uma página usando HTML5 e dê estilo profissional com CSS3, dominando Flexbox, Grid e técnicas de responsividade para que seu site fique bonito em qualquer tela.

Tópicos abordados:

Tags semânticas Flexbox Grid Tipografia Responsividade
Intermediário

Desenvolvimento Web com JAVASCRIPT

⏱️ 5 semanas
📚 Intermediário

Adicione interatividade às páginas manipulando o DOM, tratando eventos, consumindo APIs com Fetch e trabalhando com JSON; descubra como o JavaScript moderno transforma páginas estáticas em aplicações dinâmicas.

Tópicos abordados:

Sintaxe ES6+ DOM Eventos Fetch API JSON
